# Design Context

## Users

Researchers, data scientists, and meteorologists working on Thailand heatwave prediction. Users work in office/lab settings, training machine learning models, running predictions, and analyzing meteorological data. They need a reliable, professional interface for scientific workflows.

## Brand Personality

Professional + Scientific + Precise

The interface should feel like a research tool - data-focused, reliable, and trustworthy. Not playful or consumer-facing. Users expect accuracy and clarity in data visualization and model metrics.

## Aesthetic Direction

Scientific dashboard aesthetic

- Clean, data-focused interface with emphasis on clarity
- Dark mode preferred for long data analysis sessions (lab/office evening work)
- Reference: Scientific computing tools like Jupyter, TensorBoard, but more polished
- Anti-reference: Consumer apps with excessive decoration, playful animations, or marketing-style design

## Design Principles

1. Data clarity first: Visualizations and metrics must be immediately readable
2. Professional restraint: Minimal decoration, focus on functionality
3. Scientific precision: Accurate representations of data, no misleading visual flourishes
4. Efficiency: Power users should be able to navigate quickly without friction
5. Reliability: Interface should feel stable and trustworthy for critical research work
